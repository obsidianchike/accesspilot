# AccessPilot

## AI-Powered Accessibility Copilot

AccessPilot is an AI-powered accessibility auditing platform developed for the Microsoft Agents League Hackathon 2026.

The platform helps developers build more accessible websites by automatically identifying accessibility issues, explaining why they matter, and generating actionable remediation recommendations.

## Problem

Millions of websites contain accessibility barriers that make digital experiences difficult or impossible for people with disabilities.

Developers often struggle to identify accessibility violations and understand how to fix them efficiently.

## Solution

AccessPilot combines automated accessibility scanning with AI-powered reasoning to help developers:

* Detect WCAG accessibility violations
* Understand the impact of each issue
* Prioritize fixes by severity
* Generate accessibility recommendations
* Improve overall accessibility compliance

## Core Features

### Accessibility Scanner Agent

Analyzes websites and detects accessibility issues using automated testing.

### Accessibility Reasoning Agent

Explains accessibility violations and their impact on users.

### Accessibility Fix Agent

Generates recommendations and code-level remediation guidance.

### Accessibility Coach Agent

Provides educational explanations to help developers learn accessibility best practices.
